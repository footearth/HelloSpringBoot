# Hello Spring Boot

## Run

1. clean

  ```bash
  make clean
  ```

1. wrapper

  ```bash
  make wrapper
  ```

1. run spring boot

  ```bash
  make run
  ```

## Spring 生态

* 服务发现，服务注册
  * Spring Cloud Netflix Eureka
* 集中化配置中心
  * Spring Cloud Config
* API 服务网关
  * Spring Cloud Netflix Zuul
* 容错处理，断路器
  * Spring CLoud Netflix Hystrix
